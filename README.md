# 筋肉クイズアプリ - 医学部学習教材

スポーツ動作と筋肉の関係を学ぶクイズアプリです。

## 使い方

1. `index.html` をブラウザで開く
2. 表示された動作について「使わない筋肉」を選択
3. 正解/不正解が表示され、スコアが記録される

## 特徴

- **間違えやすい問題設計**: 同じ部位の筋肉から出題（例：上腕二頭筋 vs 上腕三頭筋）
- **筋肉の画像表示**: Wikipedia の解剖学画像を使用
- **24種類の動作**: 様々なスポーツをカバー

## データの追加方法

`data.js` の `sportsData` 配列に追加：

```javascript
{
    sport: "スポーツ名",
    action: "動作名",
    usedMuscles: ["使う筋肉1", "使う筋肉2", "使う筋肉3"],
    unusedMuscles: ["使わない筋肉"]
}
```

### 問題作成のポイント

- **同じ部位の筋肉を選ぶ**: 上半身の動作なら上半身の筋肉のみ
- **拮抗筋を使う**: 屈筋 vs 伸筋（例：二頭筋 vs 三頭筋）
- **機能の違いを意識**: 押す筋肉（大胸筋）vs 引く筋肉（広背筋）

## 画像の追加

`data.js` の `muscleImages` オブジェクトに追加：

```javascript
"筋肉名": "画像URL"
```

### 無料の解剖学画像ソース

- **Wikimedia Commons**: https://commons.wikimedia.org/
- **検索例**: "Biceps brachii" "Quadriceps" など英語名で検索
